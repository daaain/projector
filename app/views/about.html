Balance Projector

Project Scope

Our main goal is to create an open source personal financing app, that helps its users to project their financial situation.

Acceptance criteria / High level (long term) goals

The app should...

  • ...work offline and should be able to smart sync, when online
  • ...run outside the browser as a packaged app (built using Yeoman)
  • ...visualise the user's transactions using charts
  • ...let the users to group their transactions

Hackathon goals

  • The team wants to connect the existing app prototype to PouchDB, so we don't need to worry about what database solution the browser is using
